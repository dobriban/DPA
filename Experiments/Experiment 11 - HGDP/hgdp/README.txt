        HGDP-CEPH HUMAN GENOME DIVERSITY PANEL DATABASE SUPPLEMENT 1

        Stanford HGDP-CEPH  SNP Genotyping Data :
        Genotypes at more than 650,000 SNP loci
        determined with the Illumina BeadStation technology.


SNP Genotyping Data are organized by chromosome (chr_.map.gz and chr_.geno.gz gzip files).

* chr_.geno file contains in the first row the field names of HGDP-CEPH samples (total 1043).
  Fields are separated by tab. The missing genotype characters are "--" .

             HGDP00448  HGDP00479  HGDP00985 ......
  rs10439884    GG         GG         GG     ......
  rs2775537     AG         AA         GG     ......


* chr_.map file contains 3 fields and is a tab delimited file :
  SNPid, chromosome number and chromosome position NCBI B36.1
